Photos in this folder are wired into the HTML:
- hero.jpg            -> main hero on index + about (and profile card)
- hero-secondary.jpg  -> secondary image on homepage about-teaser
- article-1.jpg       -> article one hero image + homepage card
- article-2.jpg       -> article two hero image + homepage card
- article-3.jpg       -> article three hero image + homepage card

To swap photos: either replace the files here (keep the same filenames)
or edit the img src paths in the HTML.
